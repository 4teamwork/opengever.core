The dossier ID needs to be in following format:
"dossier-{sequence number}"


    >>> from plone.dexterity.utils import createContentInContainer
    >>> from opengever.base.interfaces import ISequenceNumber
    >>> from zope.component import getUtility
    >>> from plone.app.testing import TEST_USER_NAME, setRoles

Create a dossier:

    >>> d1 = createContentInContainer(self.folder,
    ...                               'opengever.dossier.businesscasedossier')
    >>> d1
    <DossierContainer at /plone/Members/test_user_1_/dossier-1>
    >>> d1.id
    'dossier-1'
    >>> getUtility(ISequenceNumber).get_number(d1)
    1

Create another dossier:

    >>> d2 = createContentInContainer(self.folder,
    ...                               'opengever.dossier.businesscasedossier')
    >>> d2
    <DossierContainer at /plone/Members/test_user_1_/dossier-2>
    >>> d2.id
    'dossier-2'
    >>> getUtility(ISequenceNumber).get_number(d2)
    2
