The dossier ID needs to be in following format:
"dossier-{sequence number}"


    >>> from plone.dexterity.utils import createContentInContainer
    >>> from opengever.base.interfaces import ISequenceNumber
    >>> from zope.component import getUtility

Create a dossier:

	>>> portal = layer['portal']
    >>> d1 = createContentInContainer(portal,
    ...                               'opengever.dossier.businesscasedossier')
    >>> d1
    <DossierContainer at ...>
    >>> d1.id == 'dossier-' + str(getUtility(ISequenceNumber).get_number(d1))
	True

Create another dossier:

    >>> d2 = createContentInContainer(portal,
    ...                               'opengever.dossier.businesscasedossier')
    >>> d2
    <DossierContainer at ...>
    >>> d2.id == 'dossier-' + str(getUtility(ISequenceNumber).get_number(d2))
    True
