Task
====
This test checks the resolve rules of a task.

    >>> from plone.dexterity.utils import createContentInContainer
    >>> from opengever.base.interfaces import ISequenceNumber
    >>> from zope.component import getUtility
    >>> from plone.app.testing import TEST_USER_ID, setRoles
    >>> setRoles(self.portal, TEST_USER_ID, ['Reviewer', 'Manager'])

Create a Task:

    >>> task1 = createContentInContainer(folder, 'opengever.task.task',
    ...     issuer=TEST_USER_ID, responsible=TEST_USER_ID)
    >>> task1
    <Task ...>

set task to in progress state

    >>> wft = task1.portal_workflow
    >>> wft.doActionFor(task1, 'task-transition-open-in-progress')

    >>> task2 = createContentInContainer(task1, 'opengever.task.task',
    ...    responsible=TEST_USER_ID, issuer=TEST_USER_ID)

    >>> task1.restrictedTraverse('resolve_check').render()
    False

So now we resolve the subtask and after the task resolving should also be work

    >>> wft.doActionFor(task2, 'task-transition-open-tested-and-closed')

    >>> task1.restrictedTraverse('resolve_check').render()
    True
