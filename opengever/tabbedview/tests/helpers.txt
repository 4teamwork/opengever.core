Test additional tabbedview helpers
==================================

Basic setup
-----------

We need some imports:

    >>> from opengever.tabbedview import helper

Let's create a fake object class for mocking real objects:

    >>> class FakeObject(object):
    ...
    ...     def __init__(self, **kwargs):
    ...         for key, value in kwargs.items():
    ...             setattr(self, key, value)

We also need to have brain, which we mock similar:

    >>> class FakeBrain(FakeObject):
    ...     pass


Task ID Checkbox helper (task_id_checkbox_helper)
-------------------------------------------------

Assume we have a brain:

    >>> brain = FakeBrain(title='The brain of a task',
    ...                   task_id='1')

Use the task id checkbox helper with this brain:

    >>> helper.task_id_checkbox_helper(brain, '')
    '<input class="noborder selectable" id="1" name="task_ids:list" title="Select The brain of a task" type="checkbox" value="1" />'
