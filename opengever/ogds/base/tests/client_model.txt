Create a client

    >>> from opengever.ogds.base.model.client import Client
    >>> from opengever.ogds.base.utils import create_session
    >>> session = create_session()
    >>> client = Client('client1')
    >>> client
    <Client client1>
    >>> session.add(client)

Get the client

    >>> session.query(Client).one()
    <Client client1>
