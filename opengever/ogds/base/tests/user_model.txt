Create a user

    >>> from opengever.ogds.base.model.user import User
    >>> from opengever.ogds.base.utils import create_session
    >>> session = create_session()
    >>> user = User('hugo.boss')
    >>> user
    <User hugo.boss>
    >>> session.add(user)

Get the user

    >>> session.query(User).one()
    <User hugo.boss>
