Authentification
================

L'API REST utilise Plone PAS (Pluggable Authentication System) pour l'authentification.

Cela signifie que, fondamentalement, toute méthode d'authentification supportée par un plugin PAS fonctionnera aussi avec l'API REST.

  Les détails précis du plugin d'authentification varient selon les clients et besoins. En cas de questions ou demandes de changement, veuillez s'il vous plaît contacter notre support.

Les chapitres suivants décrivent les méthodes d'authentification supportées par OneGov GEVER.
