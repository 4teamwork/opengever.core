
Ausblick
--------
Ab Q1/2018 wird zur Sitzungs- und Protokollverwaltung eine Sitzungs-App zur
Verfügung stehen. Mit dieser App können die Behördenmitglieder extern und
überall auf die Sitzungsunterlagen zugreifen. Die App wird mit einer Funktion
für persönliche Notizen ausgestattet sein.
