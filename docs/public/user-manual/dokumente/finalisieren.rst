Dokumente finalisieren
======================

Ein Dokument kann mit der Aktion "Finalisieren" finalisert werden, so dass es nicht mehr bearbeitet werden kann. Der Benutzer der ein Dokument finalisiert und Administratoren können dann mit "Überarbeiten" das Dokument zurück in den Status Entwurf setzten, so dass es wieder bearbeitet werden kann. Dieser Ablauf kann zum Beispiel verwendet werden um eine finale Version eines Dokument genehmigen zu lassen.
